---
title: "Delete a block"
sidebarTitle: "Delete a block"
description: "Sets a Block object, including page blocks, to archived: true using the ID specified."
---

Note: in the Notion UI application, this moves the block to the "Trash" where it can still be accessed and restored.To restore the block with the API, use the Update a block or Update page respectively.
