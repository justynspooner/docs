## Next up

Features and integrations we'll be working on next.

#### Integrations

- Airtable
- Slack: add support for commands and triggering on message events. Add actions for channel and team management.
- Stripe
- Google Sheets
- Twitter

#### Features

- Unified workflow run list. See all your runs across all your workflows in one place.
- Live logs. See your workflow logs in real time.
- Workflow presence. See if your workflow is connected and running.
- Error alerts. Get notified when your workflow fails or doesn't start.

#### Platform

- More robust and resilient webhook handling. We're working on a new webhook ingest system that will ensure webhook requests are never missed.
